h1. US002 Listar Plano Estratégico


*Como* usuário SIG que possui permissão no módulo "Planejamento Estratégico" 
*Quero* listar os planejamentos estratégicos 
*Para* visualizar todos os planejamentos estratégicos cadastrados.

----
 
 *Cenário 1:"Apresentar Breadcrumbs"*   

*Dado* que sistema apresenta uma navegação estruturada: _Página inicial>Nome do Módulo>Nome da funcionalidade_
*Quando* usuário clica em link apresentado na estrutura
*Então* sistema retorna aos passos anteriores, conforme escolha do usuário.

----

 *Cenário 2:"Filtrar por Sigla"*   

*Dado* que sistema apresenta filtros para listar
*Quando* usuário escolhe uma sigla na lista 
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para a sigla informada.

----

 *Cenário 3:"Filtrar por Tipo"*   

*Dado* que sistema apresenta filtros para listar
*Quando* usuário escolhe um tipo na lista, vide [[RN001_-_Definir_Campos_-_Plano_Estratégico]] .
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para o tipo informado.

----
 
 *Cenário 4:"Filtrar por Situação"*   

*Dado* que sistema apresenta filtros 
*Quando* usuário escolhe uma situação na lista gerada a partir do Cadastro do Plano Estratégico
*Então* sistema recupera e exibe somente os registros para a situação informada.


----
 
 *Cenário 5:"Filtrar por Data Inicial"*   

*Dado* que sistema apresenta filtros 
*Quando* usuário escolhe uma data inicial cadastrada para o Plano Estratégico
*Então* sistema recupera e exibe somente os registros com a data inicial informada.

----

 *Cenário 6:"Filtrar por Data Final"*   

*Dado* que sistema apresenta filtros 
*Quando* usuário escolhe uma data final cadastrada para o Plano Estratégico
*Então* sistema recupera e exibe somente os registros com a data final informada.


----

 *Cenário 7:"Limpar Filtros"*   

*Dado* que usuário preencheu ao menos um filtro exibido
*Quando* usuário clica em "Limpar"
*Então* sistema desfaz todas as seleções existentes.

----

 *Cenário 8:"Apresentar Opção Adicionar Plano"*   

*Dado* que sistema apresenta botão "Adicionar"
*Quando* usuário clica na opção Adicionar
*Então* sistema direciona usuário para Cadastro de Plano Estratégico [[Cadastrar_Plano_Estratégico]]

----

 *Cenário 9:"Apresentar Opção Detalhar Plano"*   

*Dado* que usuário cadastrou ao menos um plano estratégico
*Quando* usuário clica na opção visualizar representada pelo ícone "olho"
*Então* sistema direciona usuário para interface [[Visualizar_Plano_Estratégico]].

----

 *Cenário 10:"Apresentar Opção Editar Plano"*   

*Dado* que usuário cadastrou ao menos um plano estratégico
*Quando* usuário clica na opção editar representada pelo ícone "lápis"
*Então* sistema direciona usuário para interface [[Editar_Plano_Estratégico]].

----

 *Cenário 11:"Apresentar Opção Excluir Plano"*   

*Dado* que usuário clica na opção excluir representado pelo ícone "lixeira"
*E* que o sistema apresentou mensagem MSG002, vide [[16_-_Mensagens]] 
*Quando* usuário clica no botão excluir 
*Então* sistema apaga o registro na base de dados
*E* sistema apresenta ao usuário mensagem MSG001, vide [[16_-_Mensagens]] .


----

 *Cenário 12:"Apresentar Opção Cancelar Plano"*   

*Dado* que usuário clicou na opção excluir representado pelo ícone "lixeira"
*E* que o sistema apresentou mensagem com as opções "Cancelar" e "Excluir"
*Quando* usuário clica no botão cancelar
*Então* sistema fecha caixa de mensagem e retorna ao passo anterior.

----

 *Cenário 13:"Listar 5 Primeiros Registros por Padrão"*   

*Dado* que sistema apresenta lista de Plano Estratégico
*Quando* usuário entra na interface
*Então* sistema apresenta por padrão os 5 primeiros registros cadastrados e paginar os demais


----

 *Cenário 14:"Ordenar Alfabeticamente Resultado da Lista"*   

*Dado* que o usuário cadastrou dois ou mais planos estratégicos
*Quando* a lista for apresentada
*Então* o sistema ordena a lista alfabeticamente pelo Nome do plano estratégico.

----

 *Cenário 15:"Resultado da Listagem"*   

*Dado* que usuário cadastrou ao menos um plano estratégico
*Quando* o usuário entra na lista
*Então* o sistema exibe as informações, vide [[RN001_-_Definir_Campos_-_Plano_Estratégico]].

----

 *Cenário 16:"Apresentar Cabeçalho da Tabela por Padrão"*   

*Dado* que usuário ainda não cadastrou um plano estratégico
*Quando* o usuário entra no Listar
*Então* o sistema exibe as informações de cabeçalho por padrão
*E* mensagem
