h1. US003 Editar Plano Estratégico 

*Como* usuário SIG que possui permissão no módulo "Planejamento Estratégico" 
*Quero* editar plano estratégico
*Para* manter lista de plano estratégico atualizada.

----
 
 *Cenário 1:"Apresentar Breadcrumbs"*   

*Dado* que sistema apresenta uma navegação estruturada: _Página inicial>Nome do Módulo>Menu>Nome da funcionalidade_
*Quando* usuário clica em link apresentado na estrutura
*Então* sistema retorna aos passos anteriores, conforme escolha do usuário.

----

 *Cenário 2:"Editar Plano Estratégico"*   

*Dado* que usuário clicou na opção editar 
*E* modificou uma ou mais informações
*Quando* usuário clica em "Salvar"
*Então* sistema verifica se todos os campos obrigatórios foram preenchidos, vide [[US006_Definir_Campos_Plano_Estratégico]] 
*E* atualiza o registro do plano estratégico em um banco de dados
*E* apresenta ao usuário MSG001, vide [[15_-_Mensagens]] 

----

 *Cenário 3:"Adicionar Valor Dinamicamente"*   

*Dado* que usuário preenche o campo Valor
*Quando* usuário clica em "Adicionar Valor"
*Então* o sistema apresenta o registro na tela.

----

 *Cenário 4:"Remover Valor Dinamicamente"*   

*Dado* que adicionou ao menos um Valor na lista
*Quando* usuário clica na opção para excluir 
*Então* o sistema remove o registro da tela.

----

  *Cenário 5:"Limpar Campos"*   

*Dado* que usuário preencheu ao menos um campo
*Quando* usuário clica em "Limpar" 
*Então* sistema desfaz preenchimento no formulário
*E* mantém usuário na mesma interface.

----

 *Cenário 6:"Cancelar"*   

*Dado* que usuário preencheu nenhum, um ou mais campos
*Quando* usuário clica em "Cancelar"
*Então* sistema direciona usuário ao passo anterior de onde foi chamado.

----

 *Cenário 7:"Guardar Ação do Usuário em Histórico"*   

*Quando* usuário conclui ação de editar
*Então* sistema registra as informações em um banco de dados conforme RN006, vide [[13_-_Regras_de_Negócio]].
