h1. US002 Aprovar Solicitação Repactuação

*Como* usuário SIG que possui permissão no módulo "Planejamento Estratégico" 
*Quero* aprovar solicitação de repactuação
*Para* que o indicador seja corrigido.

----

 *Cenário 1:"Aprovar Repactuação Justificativa"*   

*Dado* que o usuário acionou a opção "Aprovar"
*E* sistema apresentou interface para inclusão da justificativa
*Quando* o usuário aciona a opção Salvar
*Então* o sistema aprova a repactuação
*E* mantém o registro da solicitação
*E* atualiza o campo Status da Solicitação para "Aprovado".
*E* apresenta mensagem MSG018, vide [[15_-_Mensagens]] 

------

 *Cenário 2:"Aprovar Repactuação Justificativa Cancelar"*   

*Dado* que o usuário acionou a opção "Aprovar"
*E* sistema apresentou interface para inclusão da justificativa
*Quando* o usuário aciona a opção Cancelar
*Então* o sistema mantém o registro da solicitação
*E* retorna para interface de solicitações, vide [[US001_Listar_Solicitação_Repactuação]] 

------

 **Cenário 3:"Guardar Ação do Usuário em Histórico"**   

**Quando** usuário conclui ação de salvar
**Então** sistema registra as informações em um banco de dados conforme RN006, vide regra [[13_-_Regras_de_Negócio]].
