h1. US002 Listar Projeto

*Como* usuário SIG que possui permissão no módulo "Gestão de Projetos" 
*Quero* listar projeto
*Para* visualizar todos os projetos cadastrados.

----
 
 *Cenário 1:"Apresentar Breadcrumbs"*   

*Dado* que sistema apresenta uma navegação estruturada: _Página inicial>Nome do Módulo>Menu>Nome da funcionalidade_
*Quando* usuário clica em link apresentado na estrutura
*Então* sistema retorna aos passos anteriores, conforme escolha do usuário.

----

 *Cenário 2:"Filtrar por Código"*   

*Dado* que sistema apresenta filtros 
*Quando* usuário escolhe um código na lista, vide [[US006_Definir_Campos_Projeto]] 
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para o código informado.

----

 *Cenário 3:"Filtrar por Nome"*   

*Dado* que sistema apresenta filtros
*Quando* usuário escolhe um nome na lista recuperada, vide [[US006_Definir_Campos_Projeto]] 
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para o nome informado.

----

 *Cenário 4:"Limpar Filtros"*   

*Dado* que usuário preencheu ao menos um filtro exibido
*Quando* usuário clica em "Limpar"
*Então* sistema desfaz todas as seleções existentes.

----

 *Cenário 5:"Apresentar Opção Adicionar Projeto"*   

*Dado* que sistema apresenta botão "Adicionar"
*Quando* usuário clica na opção Adicionar
*Então* sistema direciona usuário para interface [[US001_Incluir_Projeto]] 

---

 *Cenário 6:"Apresentar Opção Visualizar Projeto"*   

*Dado* que usuário cadastrou ao menos um projeto
*Quando* usuário clica na opção visualizar representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/472/T028-icone-visualizar-list.png!
*Então* sistema direciona usuário para interface [[US005_Visualizar_Projeto]] 

----

 *Cenário 7:"Apresentar Opção Editar Projeto"*   

*Dado* que usuário cadastrou ao menos um projeto
*Quando* usuário clica na opção editar representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/471/T029-icone-editar-list.png!
*Então* sistema direciona usuário para interface [[US003_Editar_Projeto]] 

----

 *Cenário 8:"Apresentar Opção Excluir Projeto"*   

*Dado* que usuário cadastrou ao menos um projeto
*Quando* usuário clica na opção excluir representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/473/T030-icone-lixeira-list.png!
*Então* sistema direciona usuário para interface [[US004_Excluir_Projeto]] 

----

 *Cenário 9:"Listar 10 Primeiros Registros por Padrão"*   

*Dado* que sistema apresenta lista projetos
*Quando* usuário entra na interface
*Então* sistema apresenta por padrão os 10 primeiros registros cadastrados e pagina os demais.

----

 *Cenário 10:"Ordenar Alfabeticamente Resultado da Lista"*   

*Dado* que o usuário cadastrou um ou mais projeto(s)
*Quando* a lista for apresentada
*Então* o sistema ordena a lista alfabeticamente pelo Nome do projeto.

----

 *Cenário 11:"Resultado da Listagem"*   

*Dado* que usuário cadastrou ao menos um projeto
*Quando* o usuário entra na lista
*Então* o sistema exibe as informações, vide [[US006_Definir_Campos_Projeto]] 

----

 *Cenário 12:"Apresentar Cabeçalho da Tabela por Padrão"*   

*Dado* que usuário ainda não cadastrou um projeto
*Quando* o usuário entra no Listar
*Então* o sistema exibe as informações de cabeçalho por padrão, vide [[US006_Definir_Campos_Projeto]].

----

 *Cenário 13:"Apresentar Mensagem (Tooltip) ao Passar Mouse no Visualizar"*   

*Dado* que sistema apresenta ações representadas por ícones
*Quando* usuário passa o mouse sobre a opção visualizar
*Então* sistema exibe ao usuário MSG010, vide [[15_-_Mensagens]] 

----

 *Cenário 14:"Apresentar Mensagem (Tooltip) ao Passar Mouse no Editar"*   

*Dado* que sistema apresenta ações representadas por ícones
*Quando* usuário passa o mouse sobre a opção editar
*Então* sistema exibe ao usuário MSG011, vide [[15_-_Mensagens]] 

----

 *Cenário 15:"Apresentar Mensagem (Tooltip) ao Passar Mouse no Excluir"*   

*Dado* que sistema apresenta ações representadas por ícones
*Quando* usuário passa o mouse sobre a opção excluir
*Então* sistema exibe ao usuário MSG012, vide [[15_-_Mensagens]]
