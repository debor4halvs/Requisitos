h2. US006 Definir Campos Tipo de Evento

h2. +Campos do Cadastro+

|_.Nome do Campo   |_.Tipo do Campo |_.Obrigatório|_.Observação             |_.Tabela BD               |_.Coluna BD              |_.Comentário                                                                                 |
|Código            |Alfanumérico    |Sim          |RN016                    |-                         |-                        |Refere-se ao código único gerado pelo sistema automaticamente para identificar o tipo
                                                                                                                                  de evento criado. Não pode ser editado pelo usuário.                                          |                                                                             
|Nome              |Alfanumérico    |Sim          |-                        |-                         |-                        |Refere-se ao nome do tipo de evento. Os tipos de eventos catalogados atualmente são: Processo, Projeto/Plano e Política/Programa. Servem como insumo para o correto preenchimento do formulário de Cadastro de Risco.                                   |                                              
|Descrição         |Alfanumérico    |Sim          |-                        |-                         |-                        |Refere-se a um breve relato sobre a finalidade do tipo de evento.|                                                                                                                                                                                  
|Limpar            |Botão           | -           |-                        | -                        | -                       |Permite ao usuário apagar as informações inseridas.                  | 
|Cancelar          |Botão           | -           |                         | -                        | -                       |Permite ao usuário do sistema retornar a uma tela anterior.          |            
|Salvar            |Botão           | -           |                         | -                        | -                       |Permite ao usuário do sistema gravar o registro do tipo de evento.   |
