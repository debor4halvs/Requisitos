h1. US002 Listar Feriado 


*Como* usuário SIG que possui permissão no módulo "Segurança" 
*Quero* listar os feriados
*Para* visualizar todos os feriados cadastrados.

----
 
 *Cenário 1:"Apresentar Breadcrumbs"*   

*Dado* que sistema apresenta uma navegação estruturada: _Página inicial>Nome do Módulo>Menu>Nome da funcionalidade_
*Quando* usuário clica em link apresentado na estrutura
*Então* sistema retorna aos passos anteriores, conforme escolha do usuário.

----

 *Cenário 2:"Filtrar por Nome"*   

*Dado* que sistema apresenta filtros
*Quando* usuário escolhe um nome na lista recuperada, vide [[US006_Definir_Campos_-_Feriado]] 
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para o nome informado.

----

 *Cenário 3:"Filtrar por Data"*   

*Dado* que sistema apresenta filtros
*Quando* usuário informa uma data, vide [[US006_Definir_Campos_-_Feriado]] 
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para a data informada.

----


 *Cenário 4:"Limpar Filtros"*   

*Dado* que usuário preencheu ao menos um filtro exibido
*Quando* usuário clica em "Limpar"
*Então* sistema desfaz todas as seleções existentes.

----

 *Cenário 5:"Apresentar Opção Adicionar Feriado"*   

*Dado* que sistema apresenta botão "Adicionar"
*Quando* usuário clica na opção Adicionar
*Então* sistema direciona usuário para interface [[US001_Incluir_Feriado]] 

----

 *Cenário 6:"Apresentar Opção Visualizar Feriado"*   

*Dado* que usuário cadastrou ao menos um feriado
*Quando* usuário clica na opção visualizar representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/472/T028-icone-visualizar-list.png!
*Então* sistema direciona usuário para interface [[US004_Visualizar_Feriado]] 

----

 *Cenário 7:"Apresentar Opção Editar Feriado"*   

*Dado* que usuário cadastrou ao menos um feriado
*Quando* usuário clica na opção editar representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/471/T029-icone-editar-list.png!
*Então* sistema direciona usuário para interface [[US003_Editar_Feriado]] 

----

 *Cenário 8:"Apresentar Opção Excluir Feriado"*   

*Dado* que usuário cadastrou ao menos um feriado
*Quando* usuário clica na opção editar representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/471/T029-icone-editar-list.png!
*Então* sistema direciona usuário para interface [[US003_Editar_Feriado]] 

----

 *Cenário 9:"Listar 10 Primeiros Registros por Padrão"*   

*Dado* que sistema apresenta lista de feriados
*Quando* usuário entra na interface
*Então* sistema apresenta por padrão os 10 primeiros registros cadastrados 
*E* pagina os demais.


----

 *Cenário 10:"Ordenar Resultado da Lista Por Data"*   

*Dado* que o usuário cadastrou dois ou mais feriados
*Quando* a lista for apresentada
*Então* o sistema ordena a lista pela data do evento.

----

 *Cenário 11:"Apresentar Cabeçalho da Tabela por Padrão"*   

*Dado* que usuário ainda não cadastrou um feriado
*Quando* o usuário entra no Listar
*Então* o sistema exibe as informações de cabeçalho por padrão, vide [[US006_Definir_Campos_-_Feriado]] 

----

 *Cenário 12:"Apresentar Mensagem (Tooltip) ao Passar Mouse no Visualizar"*   

*Dado* que sistema apresenta ações representadas por ícones
*Quando* usuário passa o mouse sobre a opção visualizar
*Então* sistema exibe ao usuário MSG010, vide [[15_-_Mensagens]] 

----

 *Cenário 13:"Apresentar Mensagem (Tooltip) ao Passar Mouse no Editar"*   

*Dado* que sistema apresenta ações representadas por ícones
*Quando* usuário passa o mouse sobre a opção editar
*Então* sistema exibe ao usuário MSG011, vide [[15_-_Mensagens]] 

----

 *Cenário 14:"Apresentar Mensagem (Tooltip) ao Passar Mouse no Excluir"*   

*Dado* que sistema apresenta ações representadas por ícones
*Quando* usuário passa o mouse sobre a opção excluir
*Então* sistema exibe ao usuário MSG012, vide [[15_-_Mensagens]]
