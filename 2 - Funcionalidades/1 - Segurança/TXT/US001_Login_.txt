h1. US001 Login 

**Como** usuário cadastrado no AD
**Quero** acessar a plataforma SIG
**Para** obter acesso ao(s) sistema(s) e utilizar as operações existentes.

----

 **Cenário 1:"Login Com Usuário Existente no AD"**   

**Dado** que usuário entra na interface de login
**E** informa usuário e senha, vide [[RN001_-_Login]] 
**Quando** usuário clica em "Entrar"
**Então** aplicação verifica preenchimento de campo(s) obrigatório(s), vide [[US007_Definir_Campos]] 
**E** verifica que o usuário existe no AD
**E** direciona usuário para interface que lista os sistemas da plataforma SIG
**E** apresenta somente o(s) sistema(s) autorizado(s) para o perfil autenticado.
**E** apresenta ao usuário MSG008, vide [[15_-_Mensagens]].

----

 **Cenário 2:"Login Com Usuário Inexistente no AD"**   

**Dado** que usuário entra na interface de Login
**Quando** usuário clica em "Entrar"
**Então** aplicação verifica preenchimento de campos obrigatórios, vide [[US007_Definir_Campos]] 
**E** apresenta ao usuário MSG004, vide [[15_-_Mensagens]] 

----

 **Cenário 3:"Login Usuário e/ou Senha Incorreto(s)"**   

**Dado** que usuário entra na interface de Login
**Quando** usuário clica em "Entrar"
**Então** sistema verifica preenchimento de campos obrigatórios, vide [[US007_Definir_Campos]] 
**E** verifica que dado informado está incorreto
**E** apresenta ao usuário MSG004, vide [[15_-_Mensagens]] 

----

 **Cenário 4:"Opção Esqueceu a Senha"**   

**Dado** que usuário entra na interface de Login
**Quando** usuário clica em "Esqueceu a Senha?"
**Então** sistema direciona usuário para interface, vide [[US003_Recuperar_Senha]] 

---- 

 **Cenário 5:"Opção Exibir Senha"**   

**Dado** que usuário entra na interface de Login
**Quando** usuário clica na opção representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/708/btn-olho-aberto.png!
**Então** sistema mostra a senha digitada pelo usuário.

----

 **Cenário 6:"Opção Ocultar Senha"**   

**Dado** que usuário entra na interface de Login
**Quando** usuário clica na opção representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/709/btn-olho-fechado.png!
**Então** sistema oculta a senha digitada pelo usuário.

----

 **Cenário 7:"Habilitar Botão Entrar "**   

**Dado** que usuário entra na interface de Login
**Quando** usuário preenche todos os campos obrigatórios, vide [[US007_Definir_Campos]] 
**Então** habilita o botão "Entrar" para o usuário.

----

 **Cenário 8:"Desabilitar Botão Entrar "**   

**Dado** que usuário entra na interface de Login
**Quando** usuário deixa de preencher um dos campos obrigatórios, vide [[US007_Definir_Campos]] 
**Então** sistema desabilita o botão "Entrar".

----

 **Cenário 9:"Apresentar Mensagem Informativa Para Login"**   

**Dado** que usuário entra na interface de Login
**Quando** sistema exibe as informações em tela
**Então** sistema apresenta automaticamente MSG009, vide [[15_-_Mensagens]]
