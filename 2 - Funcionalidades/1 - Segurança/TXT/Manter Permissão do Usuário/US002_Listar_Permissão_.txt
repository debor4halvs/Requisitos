h1. US002 Listar Permissão 

*Como* usuário SIG que possui permissão 
*Quero* listar as permissões 
*Para* visualizar todas as permissões cadastradas

----
 
 *Cenário 1:"Apresentar Breadcrumbs"*   

*Dado* que sistema apresenta uma navegação estruturada: _Página inicial>Nome do Módulo>Nome da funcionalidade_
*Quando* usuário clica em link apresentado na estrutura
*Então* sistema retorna aos passos anteriores, conforme escolha do usuário.

----

 *Cenário 2:"Filtrar por Nome do Perfil"*   

*Dado* que sistema apresenta filtros para listar
*Quando* usuário escolhe um nome na lista recuperada
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para o nome informado.

----

 *Cenário 3:"Filtrar por Permissão"*   

*Dado* que sistema apresenta filtros para listar
*Quando* usuário escolhe uma permissão na lista recuperada 
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para a permissão informada.

----

 
 *Cenário 4:"Limpar Filtros"*   

*Dado* que usuário preencheu ao menos um filtro exibido
*Quando* usuário clica em "Limpar"
*Então* sistema desfaz todas as seleções existentes.

----

 *Cenário 5:"Apresentar Opção Para Adicionar Permissão"*   

*Dado* que sistema apresenta botão "Adicionar"
*Quando* usuário clica na opção Adicionar
*Então* sistema direciona usuário para interface [[US001_Incluir_Permissão]] 

----

 *Cenário 7:"Apresentar Opção Visualizar Permissão"*   

*Dado* que usuário entrou no listar permissão
*E* escolheu uma permissão
*Quando* usuário clica na opção visualizar representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/472/T028-icone-visualizar-list.png!
*Então* sistema direciona usuário para interface [[US004_Visualizar_Permissão]] 

----

 *Cenário 8:"Apresentar Opção Desativar Permissão"*   

*Dado* que usuário entrou no listar permissão
*E* escolheu uma permissão
*Quando* usuário clica na opção desativar, representada pelo ícone !!
*Então* sistema direciona usuário para interface [[US008_Desativar_Permissão]] 

----

 *Cenário 9:"Apresentar Opção Ativar Permissão"*   

*Dado* que usuário entrou no listar permissão
*E* escolheu uma permissão desativada
*Quando* usuário clica na opção ativar, representada pelo ícone !!
*Então* sistema direciona usuário para interface [[US007_Ativar_Permissão]] 

----

 *Cenário 10:"Apresentar Opção Editar Permissão"*   

*Dado* que usuário cadastrou ao menos uma permissão
*Quando* usuário clica na opção editar representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/471/T029-icone-editar-list.png!
*Então* sistema direciona usuário para interface [[US003_Editar_Permissão]] 

----

 *Cenário 11:"Apresentar Opção Excluir Permissão"*   

*Dado* que usuário cadastrou ao menos uma permissão
*Quando* usuário clica na opção excluir representada pelo ícone !https://hom-redmine.cade.gov.br/attachments/download/473/T030-icone-lixeira-list.png!
*Então* sistema direciona usuário para interface [[US005_Excluir_Permissão]] 

----


 *Cenário 12:"Listar 5 Primeiros Registros por Padrão"*   

*Dado* que sistema apresenta lista de permissões
*Quando* usuário entra na interface
*Então* sistema apresenta por padrão os 5 primeiros registros cadastrados e pagina os demais.


----

 *Cenário 13:"Ordenar Das Permissões mais recentes para as mais antigas"*   

*Dado* que o sistema recuperou e listou permissões
*Quando* a lista for apresentada
*Então* o sistema ordena a lista de maneira decrescente

----

 *Cenário 14:"Apresentar Cabeçalho da Tabela por Padrão"*   

*Dado* que usuário ainda não cadastrou uma unidade
*Quando* o usuário entra no Listar
*Então* o sistema exibe as informações de cabeçalho por padrão, [[US008_Definir_Campos_Unidade]] 

----

 *Cenário 15:"Apresentar Mensagem (Tooltip) ao Passar Mouse no Visualizar"*   

*Dado* que sistema apresenta ações representadas por ícones
*Quando* usuário passa o mouse sobre a opção visualizar
*Então* sistema exibe ao usuário MSG010, vide [[15_-_Mensagens]]
