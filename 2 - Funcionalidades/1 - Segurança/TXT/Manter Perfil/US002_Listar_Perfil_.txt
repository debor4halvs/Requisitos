h1. US002 Listar Perfil 

*Como* usuário SIG que possui permissão no módulo Segurança
*Quero* listar os perfis
*Para* visualizar todos os perfis cadastrados 

----
 
 *Cenário 1:"Apresentar Breadcrumbs"*   

*Dado* que sistema apresenta uma navegação estruturada: _Página inicial>Nome do Módulo> Menu > Nome da funcionalidade
*Quando* usuário clica em link apresentado na estrutura
*Então* sistema retorna aos passos anteriores, conforme escolha do usuário.

----

 *Cenário 2:"Filtrar Por Nome da Permissão"*   

*Dado* que sistema apresenta filtros para listar
*Quando* usuário escolhe um módulo na lista recuperada, vide [[US010_Definir_Campos_Perfil]] 
*E* clica em filtrar
*Então* sistema recupera e exibe somente os registros para o módulo informado.

----

 *Cenário 2:"Filtrar Por Nome do Módulo"*   

*Dado* que sistema apresenta filtros para listar

----

 *Cenário 7:"Limpar Filtros"*   

*Dado* que usuário preencheu ao menos um filtro exibido
*Quando* usuário clica em "Limpar"
*Então* sistema desfaz todas as seleções existentes.

----

 *Cenário 8:"Apresentar Opção Montar Perfil"* 

**Dado** que usuário entra na opção Cadastrar Perfil
**Quando** usuário aciona a opção montar perfil
**Então** sistema direciona usuário para interface de permissão, vide [[US009_Montar_Perfil]] 

----

 *Cenário 9:"Apresentar Opção Visualizar Perfil"*

*Então* sistema direciona usuário para interface [[US004_Visualizar_Perfil]] 

----

 *Cenário 14:"Listar 10 Primeiros Registros por Padrão"*   

*Dado* que sistema apresenta lista 
*Quando* usuário entra na interface
*Então* sistema apresenta por padrão os 10 primeiros registros cadastrados e pagina os demais.


----

 *Cenário 15:"Ordenar Alfabeticamente Resultado da Lista"*   

*Dado* que o sistema recuperou e listou perfis
*Quando* a lista for apresentada
*Então* o sistema ordena a lista alfabeticamente pelo nome do perfil

----

 *Cenário 16:"Apresentar Cabeçalho da Tabela por Padrão"*   

*Dado* que usuário ainda não cadastrou um perfil
*Quando* o usuário entra no Listar
*Então* o sistema exibe as informações de cabeçalho por padrão, [[US010_Definir_Campos_Perfil]]  

----
